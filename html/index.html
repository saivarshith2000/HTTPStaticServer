Static HTTP Server written in single C file

What can it do ?

Runs on POSIX compliant systems

HTTP GET

Multithreading with thread pool

Single File !

How to run it ?

1. You'll need a C compiler and make

2. clone this repository and cd into it.

3. In the root of this repository, run make

4. Place your html content in the public directory and run the server executable

Command Line Arguments

Usage: %s [-p port number] [-h html directory] [-t thread pool size]
-p  Server port (Default 8000) [OPTIONAL]

-h  HTML files in this directory are served.This directory must be in the same directory as the server executable and don't add './' to the directory name! (Default 'html') [OPTIONAL]

-t  pool size. Number of threads for the server to use. (Default 8) [OPTIONAL]

-h  Shows available arguments
